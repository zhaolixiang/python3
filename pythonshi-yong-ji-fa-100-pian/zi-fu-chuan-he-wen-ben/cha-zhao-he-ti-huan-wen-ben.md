# 1、需求🙀

> 我们想对字符串中的文本做查找和替换。

# 2、解决方案😸

对于简单的文本模式，使用str.replace\(\)即可。

例如：

```
text='mark ，帅哥，18，183 帅，mark'
print(text.replace('18','19'))
print(text)
```

运行结果：

```
mark ，帅哥，19，193 帅，mark
mark ，帅哥，18，183 帅，mark
```
