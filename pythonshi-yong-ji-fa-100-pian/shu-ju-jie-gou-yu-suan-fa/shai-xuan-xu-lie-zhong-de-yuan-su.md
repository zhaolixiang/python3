# 1、需求🙀

> 序列中含有一些数据，我们需要提取其中的值或根据某些标准对序列做删减，

# 2、解决方案😸

要筛选序列中的数据，通常最简单的方法是使用列表推导式。

例如：

```
myList=[1,4,-5,10,-7,2,3,-1]
print([n for n in myList if n>0])
print([n for n in myList if n<0])
```

结果：

```
[1, 4, 10, 2, 3]
[-5, -7, -1]
```
