内置函数id\(\)可返回一个对象的标识，返回值为整数。这个整数通常对应于该对象在内存中的位置。

is运算符用于比较两个对象的标识，依次判断两个对象是否是同一个。（==用来判断值是否相等）

内置函数type\(\)则返回该对象的类型。

对象的类型本身也是一个对象，称为对象的类，并且该对象的类时唯一的，所以可以用type\(\)查询类型，然后用is来判断是否是指定类型。

实例：

```
mark=[]
if type(mark) is list:
    print("mark is list")
```

结果：

```
mark is list
```
